# my-raycast-translate

LLM APIで選択テキストを日本語に翻訳する

## 使い方

1. Raycastの拡張設定でGemini API Keyを入力します。
2. 翻訳したいテキストを他のアプリで選択します。
3. Raycastで `Translate Selected Text (Hotkey)` にホットキー（例: Cmd+J）を割り当てます。
4. テキストを選択した状態でホットキーを押すと翻訳が表示されます。

### コマンドの違い

- `Translate Selected Text (Hotkey)`：ホットキー専用。選択中テキストを取得して翻訳を起動します。
- `Translate Selected Text`：手動起動用。通常はホットキー側の利用を推奨します。
